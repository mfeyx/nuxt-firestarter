# nuxt-firestarter
A starter for Firebase with Nuxt

## .firebaserc

Put your project in here.

## Folders

- `src` = nuxt app
- `www` = all your static files that should be copied into the `public` folder (firebase hosting)

## Firebase Serve & Deploy

```shell
# serve with firebase
sh run.sh

# deploy to firebase
sh run.sh deploy
```
